My Projects



Object detection

Object detection is a powerful tool with a wide range of applications. The process involves training a machine learning algorithm on a dataset of labeled images, and then using the algorithm to identify objects within new, unlabeled images or videos.

Stock Price Prediction

Stock price prediction is the use of machine learning algorithms to analyze historical stock data and make predictions about future stock price movements. A machine learning model is trained on a dataset of historical stock data and tested to evaluate its accuracy. Once trained, the model can be used to make predictions about future stock prices to help investors make informed decisions.

Face Expression Recognizer

Face Expression Recognition using deep learning is a technique that allows computers to automatically recognize facial expressions of individuals, such as happiness, sadness, anger, or surprise. It has numerous applications in fields such as psychology, entertainment, security, and marketing.

Detecting Pneumonia from Chest X-rays

Detecting Pneumonia from Chest X-rays using deep learning is a technique that allows computers to automatically identify signs of pneumonia in medical images. It is a critical tool that can help doctors to make faster and more accurate diagnoses, leading to better patient outcomes.

Car Price Prediction

Car Price Prediction using machine learning is a technique that allows predicting the selling price of a car based on various features such as make, model, year of manufacture, mileage, condition, and others. This technology can be helpful to both car buyers and sellers as it can help to set a fair price for a vehicle.

Emotion Analysis

Emotion Analysis using machine learning is a technique that allows computers to identify and understand human emotions based on text, speech, or facial expressions. It is a powerful tool that can be used in a wide range of applications, such as customer service, market research, and mental health diagnosis.
