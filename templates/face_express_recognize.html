Real-time Face Expression Detection

Status : Yet to Deployed

The Real-time Face Expression App is a computer vision application that detects facial expressions in real-time using deep learning algorithms. It involves identifying the presence of facial expressions and their associated emotions within a video frame captured from the camera.

This project uses TensorFlow and Keras to train a deep learning model on a large dataset of images with labeled facial expressions. The model is then deployed using Flask to create a web application that can detect facial expressions in real-time.
